  <behavior>
    - apply_temperature(logits, temperature): returns logits / max(temperature, 1e-8); a near-zero temperature concentrates softmax mass on the argmax token.
    - top_k_filter(logits, k): keeps the top min(k, vocab) logits finite, masks the rest to -inf; defensive .clone() (standalone func, never mutate caller's tensor in place).
    - top_p_filter(logits, p): sort desc -> cumsum(softmax) -> shift-right mask -> always keep top-1 token; masks tokens once cumulative prob has EXCEEDED p. Keeps exactly the hand-computed nucleus on the [0.5,0.3,0.15,0.05]/p=0.8 vector (top-2).
    - next_token(logits_last, *, temperature, top_k, top_p, greedy, generator): greedy short-circuits to torch.argmax(..., dim=-1, keepdim=True) (no RNG); else composition order temperature -> top_k -> top_p -> softmax -> torch.multinomial(probs, 1, generator=generator).

    Create src/personacore/generation/sampling.py with four pure functions and a module docstring
    tagging GEN-01.

    apply_temperature(logits, temperature) -> logits / max(temperature, 1e-8) guarding division by
    zero (temperature == 0 is handled as greedy upstream in next_token, not here).

    top_k_filter(logits, k): k = min(k, logits.size(-1)); v, _ = torch.topk(logits, k); out =
    logits.clone(); out[out < v[:, [-1]]] = float("-inf"); return out. (verbatim nanoGPT idiom,
    defensive clone.)

    top_p_filter(logits, p): sorted_logits, sorted_idx = torch.sort(logits, descending=True, dim=-1);
    cum = torch.softmax(sorted_logits, dim=-1).cumsum(dim=-1); sorted_mask = cum > p; shift right
    (sorted_mask[..., 1:] = sorted_mask[..., :-1].clone(); sorted_mask[..., 0] = False) so the
    boundary token is kept and the top-1 is never masked; scatter the mask back to original order via
    sorted_mask.scatter(-1, sorted_idx, sorted_mask); return logits.masked_fill(mask, float("-inf")).

    next_token(logits_last, *, temperature=1.0, top_k=None, top_p=None, greedy=False, generator=None):
    if greedy -> return torch.argmax(logits_last, dim=-1, keepdim=True) (LongTensor, shape (1,1) for a
    (1, vocab) input, no RNG). Else: logits = apply_temperature(logits_last, temperature); if top_k is
    not None -> top_k_filter; if top_p is not None -> top_p_filter (top_k and top_p STACK); probs =
    torch.softmax(logits, dim=-1); return torch.multinomial(probs, num_samples=1, generator=generator).
    All operate on a single-sequence (1, vocab) last-position tensor — no batch (scope fence).
